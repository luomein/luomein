## Philosophy

- 實驗始於想像。

    All the experiments start from imagination.

- 實驗分為一次性實驗與重複性實驗。

    There are one-off experiments and recursive experiments.

- 重複性實驗就像在調音。

    Recursive experiments are like tuning.

- 調音使得訊號與雜訊產生區隔。

    Tuning differentiates signal from noise.
