## Web
### 實例應用

#### 從目前位置要找某座山頭的方向

- 方法一：啟用格線（ Settings -> Grid ），就會顯示某座山頭位於羅盤方位幾度

大霸尖山在我的羅盤方向233度。
![大霸尖山](./PeakFinder/大霸尖山2.png)

- 方法二：從選單 "Visible Peaks"，選擇一座山頭，畫面就會顯示你還需要轉動幾度才看得到那座山

我要左轉12度才看得到大霸尖山。
![大霸尖山](./PeakFinder/大霸尖山2.png)
